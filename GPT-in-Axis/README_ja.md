# 🌌 GPT-in-Axis（ジーピーティー・イン・アクシス）

Toggle Language: English | 日本語

**Babylon.js** と **OpenAI API** によって構築された、リアルタイムの認知可視化エンジン。
AI との会話を「論理」「感情」「抽象」の三軸に変換し、3D 空間上に視覚ノードとして配置します。

---

## 🧭 概要

**GPT-in-Axis** は、AI との対話を「二重 3D 座標空間」で可視化するフレームワークです：

- **左軸（ユーザー側）**：人間の質問を表現します。
- **右軸（AI 側）**：AI の回答を表現します。

各ノードは次の三つの意味軸によって定義された 3D 座標に配置されます：

| 軸    | 次元             | 説明                |
| ----- | ---------------- | ------------------- |
| **X** | 論理（Logic）    | 分析的 ↔ 直感的思考 |
| **Y** | 感情（Emotion）  | 冷静 ↔ 共感的感情   |
| **Z** | 抽象（Abstract） | 具体的 ↔ 比喩的認知 |

質問ノードと回答ノードを結ぶ直線は、人間と AI の思考ベクトルの**意味的整合性**を表します。

---

## 🧠 コンセプト

> 「思考とは、座標である。」

GPT-in-Axis は「推論」を幾何学的に可視化します。
すべての質問と回答は座標上の点となり、それらは**思考の星座（コグニティブ・コンステレーション）**として浮かび上がります。

---

## ⚙️ 機能

- **二重軸可視化**：ユーザーと AI が別々の認知空間を占有し、相互に接続されます。
- **リアルタイム描画**：入力から可視化完了まで約 10 秒。
- **意味スコアリング**：各ノードの論理／感情／抽象値を 0〜100%で算出。
- **インタラクティブノード**：クリックで詳細情報（テキスト・数値）を表示。
- **言語切替**：英語／日本語の二言語 UI 対応。
- **セッション管理**：会話セッションの作成・保存・読込が可能。

## 🪐 システムフロー

**ユーザー入力** → **OpenAI API** → **意味スコアリング** → **二重軸描画**
↘︎ ↙︎
**AI 回答ノード** ← **ユーザー質問ノード**

| フェーズ   | 所要時間        |
| ---------- | --------------- |
| API 処理   | 約 6〜8 秒      |
| 可視化処理 | 約 1〜2 秒      |
| **合計**   | **約 8〜10 秒** |

この遅延は、AI の思考が「形を持つ」感覚を残すために設計されたものです。
**即時性と熟考性**のバランスを取る設計思想に基づいています。

## 🚀 インストールと使い方

### 1. リポジトリをクローン

```bash
git clone https://github.com/uthuyomi/GPT-in-Axis.git
cd GPT-in-Axis
```

### 2. 依存関係をインストール

```bash
npm install
```

### 3. ローカルサーバーを起動

```bash
npm start
```

ブラウザで次の URL を開きます：
👉 **[http://localhost:8080](http://localhost:8080)**

### 4. 使用方法

1. 入力欄に質問を入力します。
2. 約 10 秒待つと、AI 処理と可視化が完了します。
3. ユーザーと AI を表す 2 つの球体が光の線で結ばれます。
4. 球体をクリックすると **infoPanel** に以下が表示されます：

   - ノードの全文（質問／回答）
   - 意味メトリクス（論理・感情・抽象％）

---

## 🎨 可視化パラメータ

| 軸               | 内容            | 範囲    | 色  |
| ---------------- | --------------- | ------- | --- |
| 論理（Logic）    | 分析的 ↔ 直感的 | 0〜100% | 青  |
| 感情（Emotion）  | 冷静 ↔ 共感的   | 0〜100% | 赤  |
| 抽象（Abstract） | 具体的 ↔ 比喩的 | 0〜100% | 紫  |

各ノードには **HUD 形式の意味ラベル** が付与され、論理・感情・抽象のスコアが色分けバーで表示されます。
スコアに応じてバーが動的に伸縮し、人間と AI の思考特性を即座に比較できます。

## 💬 実行例ワークフロー

1. **あなたが尋ねる**：「人間はなぜ夢を見るの？」
2. **GPT-in-Axis** が OpenAI API へプロンプトを送信。
3. モデルの出力をセマンティック解析：

   - 論理 = 72%
   - 感情 = 46%
   - 抽象 = 81%

4. 結果が AI 軸上に発光球として描画されます。
5. 質問と回答を結ぶ線が現れます。
